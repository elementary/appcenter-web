<p>Graph visualization based on graphviz with included a text editor and a preview area</p>
<p>Supported types:</p>
<ul>
  <li>dot</li>
  <li>neato</li>
  <li>fdp</li>
  <li>sfdp</li>
  <li>twopi</li>
</ul>
<p>Shortcuts:</p>
<ul>
  <li>F5: Compile</li>
  <li>Ctrl+N: New File</li>
  <li>Ctrl+O: Open File</li>
  <li>Ctrl+S: Save File</li>
</ul>
